# Customer Email Template

## Password Reset Assistance

**Subject:** Password Reset Assistance – FlowDesk

Hello {{Customer Name}},

Thank you for contacting FlowDesk Support.

I'm sorry to hear you're having trouble accessing your account.

To help resolve the issue, please try the following steps:

1. Click **Forgot Password** on the login page.
2. Check your inbox and spam folder for the password reset email.
3. Use the most recent password reset link you receive.
4. If the link has expired, request a new one.
5. After resetting your password, try logging in using an Incognito window or a different browser.

If you've already tried these steps and the issue still persists, simply reply to this email and we'll investigate further.

We're here to help and will work to resolve this as quickly as possible.

Kind regards,

**Product Support Team**
